# Web THR Eid Mubarak

Aplikasi web cantik untuk memberikan THR dengan tema Hari Raya Idul Fitri, tampilan mobile-friendly dan animasi menarik.

## Fitur

- Halaman sambutan dengan pesan "Happy Eid Mubarak" dan tombol "Kasih THR"
- Ketika tombol diklik, tampilkan QRIS untuk pembayaran THR
- Setelah scan QRIS, tampilkan doa dengan efek glow dan animasi confetti
- Tampilan responsif untuk perangkat mobile dengan gradient background dan animasi halus
- Font khusus untuk teks Arab dan Indonesia

## Cara Menggunakan

1. Buka file `index.html` di browser web Anda.
2. Klik tombol "Kasih THR".
3. Scan QRIS yang muncul menggunakan aplikasi e-wallet.
4. Klik "Sudah Scan" untuk melihat doa dan animasi selamat.

## Troubleshooting

- Pastikan browser Anda mendukung JavaScript.
- Jika gambar QRIS tidak muncul, pastikan file `img/qris.jpeg` ada dan path-nya benar.
- Untuk tampilan mobile, gunakan browser di perangkat mobile atau resize jendela browser.

## Teknologi

- HTML5
- CSS3
- JavaScript (ES6+)
